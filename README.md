

### Project Name

**DigiTools – platfrom**

---

### Project Description

DigiTools is a user-friendly web platform designed to showcase and manage premium digital tools in one place. Users can browse different tools, view detailed information, and add their preferred items to a cart. The project focuses on creating a smooth user experience with a modern interface and responsive design so that it works well across all devices.

---

### Technologies Used

* React.js
* Tailwind CSS
* DaisyUI
* JavaScript (ES6)
* React Toastify

---

### Key Features

**1. Interactive Product Cards**
Each digital tool is displayed using clean and informative cards that show the tool’s name, description, price, and important features.

**2. Cart Management System**
Users can easily add products to the cart, and the cart icon dynamically displays the number of selected items.

**3. Responsive and Modern Interface**
The application uses a modern layout with responsive design to ensure a smooth experience on mobile, tablet, and desktop devices.
